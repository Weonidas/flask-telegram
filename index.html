Flask App Engine Messages¶

Flask extension for working with messages using the mail & xmpp apis on App Engine.

Installing flask-gae_messages¶

Install with pip

pip install https://github.com/gregorynicholas/flask-gae_messages/tarball/master

send(to, context, method=Method.EMAIL)¶

Send a message to a designated to recipient.

Parameters:
  • to – String, email address of the recipient.
  • context – a `dict` of replacements to set for the message being sent, if one or more required paramaters for the template specified is missing a `ValueError` will be raised.
  • method – Enum of the method to send a message.
  • background – Send through a background taskqueue.
Returns:

an instance of a `taskqueue.Task` object.
flask_gae_messages.queue(to, sender, subject, body_text, body_html, method=3)¶

Enqueue a background task to send a message.

Parameters:

  • subject – String, subject of the message.
  • body_text – String, plain text body of the message.
  • body_html – String, HTML body of the message.
